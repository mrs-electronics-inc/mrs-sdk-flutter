# MRS SDK - Flutter

Welcome to the official repository for the MRS Electronic Software Development Kit for Flutter! This SDK provides developers with the tools, libraries, and documentation needed to create Flutter apps that integrate well with the MRS Electronic product ecosystem.

## Development Setup

Use `just setup` to set up your development environment.

If you use Nix, you can enter a development shell with `nix develop`. This should include all the packages required for Flutter and Android development.
