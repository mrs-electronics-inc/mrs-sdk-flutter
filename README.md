# MRS SDK Flutter

This SDK provides developers with the tools, libraries, and documentation needed to create Flutter apps that integrate well with the MRS Electronic product ecosystem.

## Installation

Add the package to your `pubspec.yaml`:

```yaml
dependencies:
  mrs_sdk_flutter: ^0.1.0
```

Then fetch dependencies:

```bash
flutter pub get
```

## Usage

Import the package in your Dart code:

```dart
import 'package:mrs_sdk_flutter/mrs_sdk_flutter.dart';
```

## Documentation

Full documentation lives at https://flutter.mrs-electronics.dev.
