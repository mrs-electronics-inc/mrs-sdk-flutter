Covered endpoints:

- `POST /loginDevice`
- `POST /api/v2/data-files`
- `POST /api/v2/data-files/{id}/file`
- `GET /api/v2/ota-files`
- `GET /api/v2/ota-files/{id}/file`
- `GET /api/v2/devices/{id}`

## Defaults

- OTA list defaults: `limit=50`, `offset=0`
- Upload input: fully buffered bytes
- Download output: raw bytes

## Retry Policy

- Retry transport errors, `429`, and `5xx`
- Delay schedule: `15s`, `30s`, `60s`
- No retry for other `4xx`
